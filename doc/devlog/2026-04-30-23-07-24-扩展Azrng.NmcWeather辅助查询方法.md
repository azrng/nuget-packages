# 扩展 Azrng.NmcWeather 辅助查询方法

## 本次目标
- 为 `Azrng.NmcWeather` 增加更高频的辅助查询方法
- 支持指定省份的城市编码列表、城市名称列表、名称编码映射
- 支持按名称快速获取省份编码和城市编码

## 核心改动
- 在 `INmcWeatherClient` / `NmcWeatherClient` 中新增：
  - `GetProvinceCodeAsync`
  - `GetProvinceNameAsync`
  - `GetProvinceCodeMapAsync`
  - `GetCityNamesByProvinceCodeAsync`
  - `GetCityNamesByProvinceNameAsync`
  - `GetCityNamesByProvinceAsync`
  - `GetCityCodesByProvinceCodeAsync`
  - `GetCityCodesByProvinceNameAsync`
  - `GetCityCodesByProvinceAsync`
  - `GetCityCodeMapByProvinceCodeAsync`
  - `GetCityCodeMapByProvinceNameAsync`
  - `GetCityCodeMapByProvinceAsync`
  - `GetCityCodeByNameAsync`
  - `GetCityNameByCodeAsync`
  - `GetCityCodeAsync`
- 新增辅助提取逻辑，统一生成去重后的城市名称列表、城市编码列表和城市名称到编码映射
- 更新 README 示例，补充辅助方法用法

## 测试补充
- 单元测试从 22 个扩展到 31 个
- 新增覆盖：
  - 省份编码/名称快捷获取
  - 省份名称到编码映射
  - 指定省份城市编码列表
  - 指定省份城市名称列表
  - 指定省份城市名称到编码映射
  - 城市名称/编码快捷转换
