# 调整日志面板鉴权配置

## 本次目标

- 移除 `DevLogDashboard` 中的 `AllowNonLocalAccess`
- 将 `AuthorizationFilter` 调整为未配置时允许匿名访问
- 支持通过可配置的 Basic 认证方案保护仪表板

## 核心改动

- 删除基于本机 IP 的访问限制，统一改为显式鉴权配置驱动
- 新增 `BasicAuthentication` 选项，支持指定认证 `Scheme` 与 `Realm`
- 中间件在配置 Basic 认证时主动调用认证方案，认证失败时返回 Basic challenge
- 补充匿名访问、自定义过滤器拒绝、Basic challenge、Basic 认证成功四类测试
- 更新 `README.md` 中的配置说明和示例
